# Chunk.js Docs

Chunk.js is the standalone native WebGL2 voxel renderer project for NiceChunk. It is developed outside the current game runtime so the renderer can be tested, profiled, and migrated safely.

Read order:

1. `architecture.md`: renderer and world-state architecture.
2. `structure.md`: folder layout and ownership.
3. `api.md`: public import surface.
4. `performance-budget.md`: mobile and draw-call constraints.
5. `migration-plan.md`: path from current terrain rendering to Chunk.js.
