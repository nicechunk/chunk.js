NiceChunk Standalone WebGL2 voxel project

Chunk.js

Standalone WebGL2 voxel renderer foundation.

This directory is split as an independent WebGL2-first voxel renderer. The current game can later import the library from src/index.js while demos, examples, debug tools, and docs stay separate.
